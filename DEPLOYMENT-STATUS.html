🌐 Deploy to Vercel (Live in 5 Minutes)

1

Create/Login to Vercel Account

Visit vercel.com and sign in with GitHub

2

Push Code to GitHub

Your local code needs to be on GitHub for Vercel to deploy

cd voci-next git add . git commit -m "VOCI Next.js - Ready for production" git push origin main
3

Import Project in Vercel

On Vercel: "Add New Project" → Select your GitHub repository → Click "Import"

4

Add Environment Variables

In Vercel Project Settings → Environment Variables, add:

NEXT_PUBLIC_FIREBASE_API_KEY=AIzaSyA3Z9nOOFBftLgq6BvotnySBbbHDJgzBnk NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=vociapp-63056.firebaseapp.com NEXT_PUBLIC_FIREBASE_PROJECT_ID=vociapp-63056 NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=vociapp-63056.firebasestorage.app NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=336825204127 NEXT_PUBLIC_FIREBASE_APP_ID=1:336825204127:web:035903ebac371854d611aa
5

Click Deploy & Wait

Vercel auto-builds and deploys your site. Takes about 3-5 minutes. Your URL will appear in the dashboard!
